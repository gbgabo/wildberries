1. Open your `settings.json`. To open the file, select `Settings` in the Windows Terminal dropdown menu while holding `Shift`.

2. Define the theme in the schemes array of your `settings.json` file:

   ```json
   {
     "name": "Wildberries",
     "background": "#19002e",
     "foreground": "#c8d3f5",
     "black": "#151515",
     "blue": "#399ee6",
     "cyan": "#00eaff",
     "green": "#00ffb7",
     "purple": "#c79bff",
     "red": "#d70040",
     "white": "#f5f5f5",
     "yellow": "#ffd700",
     "brightBlack": "#505050",
     "brightBlue": "#66baff",
     "brightCyan": "#33f1ff",
     "brightGreen": "#33ffca",
     "brightPurple": "#d6b8ff",
     "brightRed": "#ff4d94",
     "brightWhite": "#ffffff",
     "brightYellow": "#ffe34d",
     "selectionBackground": "#264f78"
   }
   ```

3. And of course, save the file and you're done!

## Info

1. Cocmic is a Wayland-Based Desktop Environment written in Rust and it's the default de for Pop_OS!

## Installation

1. Download the .ron file from this page
2. Go to Cosmic Settings -> Desktop -> Appearance -> Import
3. Select the .ron file and it should apply automatidcally

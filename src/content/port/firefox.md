## Install from Firefox Add-ons:

Install the theme directly from [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/wildberries/).

## Manage themes

As you install the theme from Firefox Add-ons, the theme is automatically applied to the browser. But if you want to manage all installed themes, go to `Settings > Extensions & Themes`, or simply type on the URL bar `about:addons` and go to `Themes`.
